隐私政策

龙虾助手 - OpenClaw Helper

概述

龙虾助手（OpenClaw Helper）是一款浏览器扩展，为 OpenClaw 聊天界面提供增强控制面板。我们高度重视您的隐私。

数据收集

本扩展 不收集 任何用户个人数据，包括但不限于：

  • 个人身份信息（姓名、邮箱、电话等）
  • 浏览历史或搜索记录
  • Cookie 或登录凭据
  • 地理位置信息

数据传输

本扩展 不向任何第三方传输数据。所有操作均在您的浏览器本地执行，扩展仅与您自己部署的 OpenClaw 服务实例通信。

本地存储

本扩展使用浏览器的 localStorage 存储少量配置偏好（如当前选择的模型、思考深度等级），这些数据仅保存在您的本地浏览器中，不会上传至任何服务器。

权限说明

本扩展请求 <all_urls> 权限，原因如下：

  • OpenClaw 可部署在用户自定义的任意域名或 IP 地址上
  • 扩展需要在页面加载后检测是否存在 <openclaw-app> 元素
  • 检测脚本运行在隔离环境（ISOLATED world），不读取、不修改非 OpenClaw 页面的任何内容
  • 仅当检测到 OpenClaw 页面时，才会激活业务功能

变更通知

如本隐私政策有更新，我们将在此页面发布最新版本。

Privacy Policy

龙虾助手 - OpenClaw Helper

Overview

OpenClaw Helper is a browser extension that provides an enhanced control panel for the OpenClaw chat interface. We take your privacy seriously.

Data Collection

This extension does not collect any personal data, including but not limited to:

  • Personal identification information (name, email, phone, etc.)
  • Browsing history or search records
  • Cookies or login credentials
  • Geolocation data

Data Transmission

This extension does not transmit any data to third parties. All operations are performed locally in your browser. The extension only communicates with your own self-hosted OpenClaw service instance.

Local Storage

This extension uses the browser's localStorage to save a small amount of configuration preferences (such as current model selection, thinking depth level). This data is stored only in your local browser and is never uploaded to any server.

Permissions

This extension requests the <all_urls> permission for the following reasons:

  • OpenClaw can be deployed on any custom domain or IP address chosen by the user
  • The extension needs to detect whether an <openclaw-app> element exists after page load
  • The detection script runs in an isolated environment (ISOLATED world) and does not read or modify any content on non-OpenClaw pages
  • Business functionality is only activated when an OpenClaw page is detected

Changes

If this privacy policy is updated, we will publish the latest version on this page.
